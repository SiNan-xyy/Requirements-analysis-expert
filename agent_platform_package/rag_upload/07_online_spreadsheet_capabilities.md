# 腾讯文档与在线表格能力说明

本材料用于 RAG 检索，帮助 Agent 在涉及腾讯文档、在线表格、飞书表格、企业微信表格、WPS 在线表格等场景时，生成更贴近实际搭建的说明。本材料强调业务能力和确认事项，不生成具体参数。

## 适用范围

适用于以下需求：

- 把采集结果写入腾讯文档。
- 从在线表格读取配置、名单、字段映射或日报模板。
- 按指定行列或单元格写入数据。
- 读取在线表格中的已有数据并进行核对。
- 生成日报、周报、月报、异常清单或处理记录。

## 常见能力

### 打开在线表格

机器人可以通过网页方式打开腾讯文档或其他在线表格。

需要确认：

- 文档链接是否固定。
- 是否需要登录。
- 是否需要扫码、短信、设备确认或二次验证。
- 账号是否有查看和编辑权限。

风险：

- 扫码登录通常不适合完全无人值守。
- 文档链接变化会导致流程无法进入目标页面。

### 定位目标 Sheet

机器人可以进入指定 Sheet 或工作表页。

需要确认：

- Sheet 名称是否固定。
- 是否存在新增、删除、重命名或隐藏 Sheet。
- 汇总页和明细页是否分开。

风险：

- Sheet 名称变化会导致写入位置错误。
- 多个 Sheet 名称相似时需要明确规则。

### 读取指定行、列或单元格

机器人可以读取表格中的指定行、指定列、单元格或表格区域。

需要确认：

- 读取区域是否固定。
- 是否通过日期、平台、店铺、字段名来定位行列。
- 表头位置是否固定。
- 是否存在合并单元格、隐藏行列或筛选状态。

风险：

- 模板结构变化会导致读取错位。
- 空值需要区分“真实为空”和“读取失败”。

### 写入指定行、列或单元格

机器人可以将数据写入指定位置。

需要确认：

- 写入位置是固定坐标，还是按日期或字段名动态定位。
- 是否允许覆盖原有内容。
- 是否需要追加新行。
- 写入前是否需要清空旧数据。
- 是否需要保留写入前的原始值。

风险：

- 单元格坐标错误会导致数据写到错误位置。
- 日期行定位错误会导致日报错位。
- 模板结构变化会导致写入失败。

### 校验写入结果

机器人可以在写入后读取目标位置，确认写入值是否与源值一致。

需要确认：

- 是否要求自动核对。
- 核对字段有哪些。
- 数字格式、千分位、百分号、小数位如何处理。
- 不一致时是否停止、通知或进入人工复核。

风险：

- 只判断“写入成功”不能证明数据正确。
- 目标表格可能自动格式化，导致源值和目标显示值不同。

## 必须向客户确认的问题

1. 在线表格链接是否固定？
2. 登录方式是否稳定，是否需要扫码或二次验证？
3. 机器人账号是否有编辑权限？
4. 目标 Sheet 名称是否固定？
5. 写入位置是固定单元格、按日期定位，还是按字段名定位？
6. 是否需要读取目标表格已有数据？
7. 是否需要写入后自动核对？
8. 模板结构变化时是否停止执行并通知人工？
9. 部分数据写入失败时，是否允许保留已成功部分？

## 报告表达建议

对客版可以写：

“机器人可打开在线表格，并按确认好的 Sheet、行列或单元格位置写入日报数据。为避免写错位置，需要先确认模板结构、登录方式和写入规则。”

实施对齐版可以写：

“可参考的影刀能力包括：打开网页、定位在线表格页面、读取表格区域、按行列写入数据、写入后读取校验、记录异常清单。具体元素和参数需在实际环境中确认。”

## 不应写成确定事实的内容

以下内容未确认前只能写成待确认项：

- 腾讯文档一定可以无人值守登录。
- 目标 Sheet 一定固定。
- 单元格坐标一定不变。
- 写入成功就代表数据正确。
- 扫码登录可以自动处理。
